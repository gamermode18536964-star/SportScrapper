# SportScrapper

## What is this?
This website is very easy to use, you can enter in filters to get information about NFL, NBA, MLB, NHL, NCAAF, NCAAB, WNBA games. Everyday at 10 AM the data automaticaly updates to contain the most recent games, and allows you to export it into a easy to use CSV file.
